# DP Sliding Window

- Reduces memory usage
  - only storing the states that are needed
- Only works with iterative DP

---

<Head>
    <title>Hello</title>
</Head>


<CodeSurfer theme={vsDark}>

  ```cpp
  int dp[n]; dp[0] = 0, dp[1] = 1;
  for (int i = 2; i < n; i++) {
    dp[i] = dp[i-1] + dp[i-2];
  }
  cout << dp[n-1] << endl;
  ```

</CodeSurfer>

<ListSteps>

- Memory: `O(n)`
- When calculating `i`, we only need `i-1` and `i-2`!

</ListSteps>

### Fibonacci (Sliding Window)

<CodeSurfer theme={vsDark}>

  ```cpp
  int dp[2]; dp[0] = 0, dp[1] = 1;
  for (int i = 2; i < n; i++) {
      dp[i%2] = dp[(i-1)%2] + dp[(i-2)%2];
  }
  cout << dp[(n-1)%2] << endl;
  ```

</CodeSurfer>

<ListSteps>

- Memory: $O(1)$

</ListSteps>
